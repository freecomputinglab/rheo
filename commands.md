# Example rheo commands

```bash
rheo compile examples/phd_thesis
# should compile HTML and PDF from the .typ files in the folder 
```


```bash
rheo compile --html examples/phd_thesis
# should compile only HTML from the .typ files in the folder 
```


```bash
rheo compile --pdf examples/phd_thesis
# should compile only PDF from the .typ files in the folder 
```
